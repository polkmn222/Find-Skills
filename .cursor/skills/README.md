# Cursor Skills

Installed skill:

```text
.cursor/skills/find-skills/SKILL.md
```

Use in Cursor:

```text
@find-skills find skills for this project.
```

For an idea:

```text
@find-skills find skills for a CRM web project.
```

The skill is Markdown-only. It reads `SKILL.md` files and does not require Python.
