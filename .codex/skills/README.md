# Codex Skills

Installed skill:

```text
.codex/skills/find-skills/SKILL.md
```

Use in Codex:

```text
Use find-skills to find skills for this project.
```

For an idea:

```text
Use find-skills to find skills for a small game project.
```

The skill is Markdown-only. It reads `SKILL.md` files and does not require Python.
