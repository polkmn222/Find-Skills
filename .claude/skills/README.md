# Claude Code Skills

Installed skill:

```text
.claude/skills/find-skills/SKILL.md
```

Use in Claude Code:

```text
Use find-skills to find skills for this project.
```

For an idea:

```text
Use find-skills to find skills for a CRM web project.
```

The skill is Markdown-only. It reads `SKILL.md` files and does not require Python.
