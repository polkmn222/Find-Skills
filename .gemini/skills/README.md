# Gemini Skills

Installed skill:

```text
.gemini/skills/find-skills/SKILL.md
```

Use in Gemini CLI:

```text
Use find-skills to find skills for this project.
```

For an idea:

```text
Use find-skills to find skills for an agent project.
```

The skill is Markdown-only. It reads `SKILL.md` files and does not require Python.
